-Um projeto sobre login no Google, com cadastro. 

-Utilizamos o TypeScript(ts e tsx), CSS e o .env para a chave do login.

-Os desenvolvedores são Isaac Barros e Alexandre Carvalho esudantes da FAETEC no curso de informática.

-Para usar é simples, na Home vai estar o login no Google, no register vai ter um formulário(Nome, Email, Telefone e Idade), e no About vai falar um pouco sobre os desenvolvedores do site.
Instruções para execução local;
